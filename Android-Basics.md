# Android Basics

## What is App Development?
App development is the process of creating software applications for mobile or desktop.

## Platforms
- **Android** → Uses Java/Kotlin
- **iOS** → Uses Swift
- **Cross-platform** → Flutter (Dart), React Native (JavaScript)

## Tools Needed
- Android Studio (IDE)
- Java Development Kit (JDK)
- Emulator or physical device for testing
