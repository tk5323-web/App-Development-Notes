# UI Design Notes

## What is UI?
UI (User Interface) is how the app looks and interacts with the user.

## Layouts in Android
- XML files define the screen design.
- Common elements:
  - **TextView** → Display text
  - **Button** → Click actions
  - **EditText** → Input field
  - **ImageView** → Show images

## Example
A simple layout with a button and text:
```xml
<TextView android:text="Hello World!" />
<Button android:text="Click Me" />
